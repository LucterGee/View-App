## 注意点
### 1.打包后，注意打包的类型
 
在webpack.config.js中操作

将编译的代码改成非调试模式，这条推荐修改

```javascript
    devtool: 'source-map',//打包时，注释掉这行
    // devtool: false,//打包时，解注释这行
```

将打包模式为生产模式，进行混淆
```javascript
    mode: "development"//运行环境：开发环境
    // mode: "production",//运行环境：生产环境
```
